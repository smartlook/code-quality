# @smartlook/eslint-config-prettier

This config should be used in all projects which use Prettier.

## Installation

```sh
npm i -D @smartlook/eslint-config-prettier
```

```js
// .eslintrc.js

module.exports = {
    extends: ['@smartlook/eslint-config-prettier'],
}
```
